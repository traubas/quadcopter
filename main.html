Autonomous QuadCopterBy Judy And Ofir

EPISODE IV
A NEW HOPE FOR AUTONOMOUS QUADCOPTERS!

Our Goal: Creating an infrastructure for embedding image proccessing algorithms

The idea of the project is to get multiple quadcopters fly autonomously in a group maintaining a structure in which they do not collide with each other.

The algorithm for the image processing exists and so in this project we construct an infrastructure for the flight control.



Quadcopter in use: Syma X5C Explorers



Single Board computers used: Raspberry Pi Zero W x3



Cameras: Raspberry Pi Camera



Communication with the quadcopter: Nrf Transceiver
